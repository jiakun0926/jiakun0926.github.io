自己整理的ES6基础知识
2018-12-19|笔记
阅读量:

ES6箭头函数这一块在这里就不写了,这个东西个人觉得没必要从百度上找一些 层层嵌套的结构最后问你this指向的题来刷,没什么意义,一是我在工作中很少遇到那些层层嵌套的结构,再者说this指向谁我打印一下不就知道了,反正我是觉得箭头函数这一块没什么太大的必要去记忆一些东西,用的多了,有些东西不言而喻的就明白了

函数的默认值

ES6中支持在函数的参数中 以 key = value (a = 20) 的形式来定义一个默认值

var foo = (a,b=20) =>{
    console.log(a+b)
}
// 这里 参数b设置了一个默认值为20 如果只传一个参数a 参数b会以20的值来参与到函数当中
foo(20); // 40
// 这里如果传了参数b(40) 则取代了之前的默认值(20)
foo(10,40); // 50

解构赋值

解构赋值简单的字面理解就是 把一个数组或者对象先解构,后赋值

数组的解构赋值

// 数组解构对应关系 与下标有关  对应不了的值为undefined
var arr = [1,2,3,4,5]
var [a,b, ,c,d] = arr
console.log(a,b,c,d) // 1 2 4 5

var [a,b,c,d] = arr
console.log(a,b,c,d) // 1 2 3 4 undefined

// 另一种数组解构赋值形式(我反正基本用不到)
var arr2 =  [ 0 , 1, 2 , [ 3 , 4 ,[ 5 ] ] ];
var k = arr2[3][2][0]
var [ , , ,[ , ,[g]]] = arr2
console.log(k,g) // 5

对象的解构赋值

// 对象解构定义的变量与属性名称有关 与下标没有关系
var obj = {
    name: "张三",
    age: 27,
    say:() => {
        console.log(1)
    }
}
// 这里定义的变量必须与对象中的key相同
var {name,age,say} = obj
console.log(name) // 张三
console.log(typeof(say))  // function

(…)剩余运算符

// 常用于拼接数组 [...arr,...arr2]
var arr1 = [1,2]
var arr2 = [2,3]

// 未学习ES6时用concat方法来拼接数组
var a  = arr1.concat(arr2)
console.log(a) // 1 2 2 3

// 直接用ES6的剩余运算符来拼接数组
var b = [...arr1 , ...arr2]
console.log(b) // 1 2 2 3

// ...数组
var arr3 = [1,9,4]
console.log(...arr3) // ...arr3 打印 1 9 4三个独立的数字  不是数组了

// 如果你直接...一个对象会报错的
var obj ={
    a:1,
    b:2,
    c:3
}    
console.log(...obj); // 报错
console.log({...obj}); // 这才是正确姿势

Array.from

作用就是把类数组变为数组 使其可以使用数组的方法

function a(a,b){
    console.log(arguments) // arguments会得到一个包含所有参数的伪数组
    var arr = Array.from(arguments)
    // 这里就可以用数组的push方法了
    arr.push(3)
    console.log(arr) // 数组
}
a(1,2)

set数据结构

ES6的一种数据结构,里边的所有成员都是唯一的,结构可以记不住,Set去重方法一定要记住,非常好用

  • size属性 代表大小 相当于数组的length
  • set数据结构中的成员都是唯一的 因此常用作数组去重
  • set.add(n) 在数据结构中添加n 如果出现重复则不添加
  • set.delete(n) 删除n
  • set.clear(); 清除set里边所有的
var set = new Set();
set.add(1)
set.add(3)
set.add(1)
// 由于成员的唯一性 上边添加了两个1 打印结果也只显示1个1
console.log(set) // 1,3

//去重方法
var arr = [ 1,3,4,4,7,1,3]
console.log(new Set(arr)) // 1 3 4 7
