Swiper中的箭头函数

ES6很多东西还是非常好用的,对我而言在工作中最直观的感受就是可以改变原来的this指向
比方说Vue项目中使用Swiper,在Swiper的事件函数中this的指向是Swiper实例对象,但是用ES6的箭头函数this的指向会改变为Vue实例,由于Swiper大部分事件都是在实例对象中写的,因此没法用 var that = this 的方式预先保存Vue实例,因此箭头函数还是挺好用的,如下图我的一个Vue项目中的Swiper实例
